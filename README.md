# Email Signature Builder — Maria Belevitskaya

Interactive email signature builder with customizable:
- 🎨 **8 color themes** (Warm, Ocean, Forest, Midnight, Rosé, Mono, Terracotta, Lavender)
- 🔤 **6 font pairings** (DM Sans, Playfair+Manrope, Cormorant+Outfit, Syne, Archivo, Source Serif)
- 📐 **4 layouts** (Horizontal, Stacked, Compact, Bold)
- 📝 **2 content modes** (Minimal / Full with social icons & tagline)
- ➕ **Dynamic fields** (add/remove phone, email, address, website, custom)

## Usage

1. Open `index.html` in a browser (or visit the hosted version)
2. Customize the design using the control panel
3. Click editable fields to enter your information
4. Click **Copy Signature** → paste into Gmail / Outlook settings
